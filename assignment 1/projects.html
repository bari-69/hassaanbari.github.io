My Projects

Lahore Transport Management System

A robust SQL-based solution to streamline Lahore’s metro and bus networks. It manages passenger data, route optimization, driver scheduling, and ticketing, providing real-time updates for traffic and maintenance. Built with a 15-table normalized MySQL structure, LTMS ensures data integrity and scalability, enhancing passenger experience with seamless ticketing and feedback features.

MIPS Assembly Language Project

A project for COMP 300 Spring 2024 focused on mastering MIPS assembly programming. It involves four key tasks: reading/writing text files, analyzing file content to count lines, words, and characters, searching for specific strings, and removing blank lines. The project emphasizes self-learning, efficient file manipulation, and algorithm development.

COAL Final Project
